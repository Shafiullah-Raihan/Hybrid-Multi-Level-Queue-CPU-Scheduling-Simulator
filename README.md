# Hybrid Multi-Level Queue CPU Scheduling Simulator

A modern and interactive **CPU Scheduling Simulator** built with **HTML, CSS, and JavaScript**.  
This project demonstrates how different CPU scheduling algorithms work inside a **Hybrid Multi-Level Queue Scheduling System** with real-time visualization, process management, and performance analysis.
